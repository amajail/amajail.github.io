### Agent Management Platform
**Client: Big Four company (confidential) · CloudX · 2024–2025**
badge: [Featured]

consuming MCP tools and OpenAPI plugins. Built to scale across multiple business units
with full enterprise integration.

- RAG pipeline orchestration with MCP and OpenAPI plugin consumption
- Real-time agent status via SignalR

- Secure identity management with Microsoft Entra
- Event-driven architecture with Azure Service Bus

App Configuration · SignalR · Terraform · GitHub Actions · Microsoft Entra
