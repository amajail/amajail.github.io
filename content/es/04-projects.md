### Agent Management Platform
**Cliente: Big Four (confidencial) · CloudX · 2024–2025**
badge: [Destacado]

Plataforma enterprise para gestión de agentes de IA que realizan Retrieval-Augmented Generation (RAG),
consumiendo herramientas MCP y plugins OpenAPI. Construida para escalar en múltiples unidades
de negocio con integración enterprise completa.

Destacados:
- Orquestación de pipelines RAG con MCP y plugins OpenAPI
- Estado de agentes en tiempo real vía SignalR
- Entrega multi-equipo en 5 equipos de 4 países
- Gestión de identidad segura con Microsoft Entra
- Arquitectura event-driven con Azure Service Bus

Stack: React · PostgreSQL · Azure Cosmos DB · AKS · Azure Functions · Service Bus ·
App Configuration · SignalR · Terraform · GitHub Actions · Microsoft Entra

### Plataforma de Impuestos Enterprise
**Cliente: Big Four (confidencial) · Cognizant Softvision · 2022–2024**

Plataforma de gestión tributaria a gran escala para una empresa global. Arquitectura de microservicios
con integración completa en Azure, reportes Power BI y hardening de seguridad con SonarQube.

Destacados:
- SAFe PI planning en equipos multiculturales
- Redis para acceso a datos de alta performance
- Calidad de código y seguridad con SonarQube
- Integración con Power BI para reportes ejecutivos

Stack: .NET · REST APIs · Azure Functions · Service Bus · Redis · SQL · Power BI · SonarQube

### Plataforma Enterprise Universitaria
**Cliente: UADE · 2009–2020**

Plataforma de larga duración para más de 20.000 usuarios. Evolucionó de un monolito .NET
a un sistema escalable en Azure con CI/CD automatizado, integración de pagos y manejo de alta concurrencia.

Destacados:
- Escalado para alta concurrencia en Azure
- CI/CD automatizado con Jenkins
- Oracle PL/SQL: procedimientos, paquetes y funciones
- Integración con pasarela de pagos

Stack: .NET 4.8 · Oracle (PL/SQL) · Azure · Jenkins · REST APIs
